👋 About Me

I am a Full-Stack Web Developer and AI Enthusiast, currently pursuing a Master's degree in Intelligent Systems for Education.

My journey combines solid experience in web development, Big Data pipelines, and AI integration, with a focus on building real-world solutions that are scalable, efficient, and intelligent.

🌍 What I Do

Web Development (Frontend & Backend):

I develop modern, responsive web applications using technologies like React, Next.js, JavaScript, Node.js, and Spring Boot.

My backend experience includes working with RESTful APIs, JWT authentication, MySQL/PostgreSQL, and building robust microservices architectures.

Data Engineering & Big Data:

I work on data pipelines with tools like Apache Spark, Hive, HDFS, and Docker, handling large datasets for anomaly detection and real-time processing.

I have implemented Structured Streaming pipelines and integrated Spark MLlib for machine learning tasks.

AI & NLP Integration:

I integrate AI models into web applications to automate tasks like email generation, sentiment analysis, and natural language interactions.

I have hands-on experience with LSTM, RNN, DistilBERT, and NLP APIs for real-world use cases.

Educational Technology Projects:

Passionate about using technology for education, I develop systems that combine intelligent recommendations, content analysis, and interactive dashboards to improve the learning experience.
